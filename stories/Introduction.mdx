# Chiti UDS — Storybook

This is the component documentation for **Chiti Unified Design System (UDS) V3**.

## Quick Start

The design system provides:

- **Design Tokens** — CSS variables for colors, spacing, typography
- **Components** — Reusable UI components (Buttons, Inputs, etc.)
- **Patterns** — Common UI patterns (Forms, Tables)

## Available Components

| Component | Description |
|----------|-------------|
| [Button](/?path=/story/components-button--all-variants) | Action triggers with multiple variants |
| [Input](/?path=/story/components-input--all-inputs) | Text inputs and selects |
| [Toggle](/?path=/story/components-toggle--all-toggles) | Switch controls |
| [Badge](/?path=/story/components-badge--all-badges) | Status indicators |
| [Tabs](/?path=/story/components-tabs--default) | Tabbed navigation |
| [Card](/?path=/story/components-card--default) | Container components |

## Theme Support

All components support **Dark** and **Light** themes via CSS custom properties.

## Usage

Copy the CSS from `design_system_showcase_v3.html` and use the component classes:

```html
<button class="c-btn primary">Click Me</button>
<input class="c-input" type="text" placeholder="Enter value">
<span class="c-badge success">Active</span>
```

## 📏 4. Spatial Alignment System
All Chiti V3 Layouts must adhere to the **Horizontal Baseline**.

- **Vertical Rhythm**: Sibling components (e.g., Input to Input) must be separated by `space-4` (24px).
- **Grid Density**:
